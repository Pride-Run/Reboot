Welcome to Pride Run

Chronic Foundation is non profit organisation, organizing an event Pride Run 2019

Cause – Run for Promoting "GREEN ENERGY".

It is the time to think how each one can address the climate change. Energy is one of the key sector in contributing the carbon. In this PRIDE RUN, we are campaigning to create awareness of Green energy ( Solar, Wind, Bio waste etc.,) and fast track Zero-Carbon growth.
